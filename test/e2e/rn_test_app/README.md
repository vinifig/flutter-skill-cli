# Flutter Skill — React Native Test App

React Native app for E2E testing the Flutter Skill React Native SDK.

## Features

- Counter with increment/decrement
- Text input with submit
- Switch toggle
- FlatList with 20 items
- Navigation (state-based) to Detail and Form screens
- Form with name/email fields

## Run

```bash
npm install
npx react-native run-android  # or run-ios
```

All elements have `accessibilityLabel` set for selector-based targeting.
