# Skills Submission

This folder contains the skill definition for submitting to various skill registries.

## Anthropic Skills Repository

Submit to the official Anthropic skills repository:

1. Fork https://github.com/anthropics/skills
2. Copy `flutter-skill/` folder to `skills/Development & Technical/`
3. Submit a Pull Request

## Awesome Claude Skills

Submit to the community-maintained list:

1. Fork https://github.com/travisvn/awesome-claude-skills
2. Add entry to README:
   ```markdown
   | [flutter-skill](https://github.com/ai-dashboad/flutter-skill) | Control Flutter apps - inspect UI, perform gestures, take screenshots |
   ```
3. Submit a Pull Request

## Awesome MCP Servers

Submit to the MCP servers list:

1. Fork https://github.com/punkpeye/awesome-mcp-servers
2. Add entry to README under Mobile/Desktop:
   ```markdown
   | [flutter-skill](https://github.com/ai-dashboad/flutter-skill) | Control Flutter apps via Dart VM Service - inspect, tap, scroll, screenshot |
   ```
3. Submit a Pull Request
