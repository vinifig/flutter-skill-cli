Flutter Skill - AI Agent Bridge for Flutter Apps

Give your AI Agent eyes and hands inside your Flutter app.

Flutter Skill creates a bi-directional control channel between AI coding assistants (Claude Code, Cursor, Windsurf, etc.) and running Flutter applications through the Dart VM Service Protocol.

Key Features

UI Inspection

  • Widget Tree Analysis - Explore the complete widget hierarchy with depth control
  • Interactive Elements - Find buttons, text fields, and other actionable widgets
  • Element Properties - Get size, position, visibility, and state information
  • Text Extraction - Extract all visible text content from the screen

Interactions

  • Tap & Click - Single tap, double tap, and long press gestures
  • Text Entry - Enter text into text fields programmatically
  • Gestures - Swipe, drag, and scroll operations
  • Navigation - Go back, view navigation stack, get current route

State & Validation

  • State Queries - Get text field values, checkbox states, slider values
  • Wait Conditions - Wait for elements to appear or disappear
  • Screenshots - Capture full app or specific element screenshots

Development Tools

  • Hot Reload - Trigger hot reload remotely
  • Logs & Errors - View application logs and error messages
  • Performance - Monitor performance metrics

Zero Configuration Setup

Just run flutter_skill launch /path/to/project and the tool automatically:

  1. Adds the flutter_skill dependency if missing
  2. Patches main.dart with initialization code
  3. Launches the app and captures the VM Service URI

MCP Server Support

Works seamlessly with MCP-compatible AI agents. Add to your MCP configuration:

{
  "flutter-skill": {
    "command": "flutter-skill",
    "args": ["server"]
  }
}
